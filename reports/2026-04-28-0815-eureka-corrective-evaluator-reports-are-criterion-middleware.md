# /eureka — corrective evaluator reports can become criterion middleware

*Generated 2026-04-28 08:15. This run opened to test whether the threshold-crossing
pattern named in copy middleware and interaction middleware also existed in
evaluator/report arcs, or whether interaction work was the natural ceiling. The
answer is yes, but with a different steering object again.*

## The discovery

Some corrective evaluator reports cross from **after-the-fact interpretation**
into **criterion middleware**.

This happens when a report does not merely say "the result was wrong."

It says:

- this experiment measured cold-baseline propensity, not capacity
- this rubric excluded the very register it claimed to test
- this evidence tier supports a narrower claim than the headline implied

At that point the report is no longer just cleaning up the past.

It is governing what later experiments are allowed to claim.

## Why this is different from the other middleware categories

The project now has three already-named steering shapes:

- **trajectory middleware** steers what gets written next at the session arc
  level
- **copy middleware** steers the next sentence
- **interaction middleware** steers the next control-shape move

This fourth shape steers something else:

- **criterion middleware** steers what later evidence is allowed to mean

Its steering object is not prose and not interaction.

It is the interpretation frame of the instrument itself.

## The threshold-crossing condition

Not every correction report becomes criterion middleware.

The threshold is narrower:

- the report identifies a mismatch between the instrument and the claim
- the mismatch changes later rubric design, cell design, or evidence wording
- later work would be misleading if it ignored the correction

When those conditions hold, the report is functioning as live methodology
infrastructure.

## Why this matters

Without naming this category, the repo risks treating the most important
methodological corrections as mere cautionary reading.

But these reports were already doing more than warning.

They were silently governing later honesty:

- what counts as capacity evidence
- what counts as a valid rubric boundary
- what kind of headline a given cell structure can support

Once that is true, they should be treated like live steering surfaces, not
optional background reading.

## Reading

The deeper pattern is now sharper than "summaries steer things."

Different report families steer different layers of the work:

- some steer prose
- some steer interaction
- some steer trajectory
- some steer the criteria by which later evidence gets interpreted

The evaluator/report arc does show the same threshold-crossing pattern.

But it does so at the level of **what later judgment is allowed to say**.

That is its own category, and the correction reports earned it by repeatedly
changing future honest methodology rather than merely revising past rhetoric.
