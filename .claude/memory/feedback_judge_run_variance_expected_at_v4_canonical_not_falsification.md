## Rule

When v4-canonical scoring of cosmology bench-work returns
substrate-distinctness magnitudes lower than prior cross-check or
codex-blessed claim language, **judge-run-variance within 30-40pp on
N=3 cells is the expected finding shape** — not falsification of the
underlying claim. Threshold-clearing (≥30pp gap per Move-15) matters
more than magnitude-precision-matching to original framing.

**Why:** v4 scorer with fixed cap-scope-guardrail (codex 8th note 3,
post-bug-fix 2026-05-09) reads cells more strictly on borderline
cases than prior J3 cross-check or J3+J2 dual-judge work. The
strictness is canonical-instrument-correct, not over-strict; the
magnitude variance reflects the load-bearing instrument doing its
job + reasonable judge stochasticity at N=3 sample size. Forcing
magnitude-precision-matching would either:
(a) Over-fit the rubric to the original measurement run (anti-honesty)
(b) Manufacture false-precision in pre-registered thresholds
(c) Inflate Sapphire-firing claims beyond what threshold-clearing
    actually warrants

**How to apply:**

1. **Pre-registered threshold is the verdict-gate, not the magnitude.**
   Move-15's ≥30pp pass-rate gap is the load-bearing falsification-
   condition-1 trigger. If gap ≥30pp, signal holds; if <30pp, narrow
   or refuse. Magnitudes above 30pp are honest-data variance.

2. **Document magnitude variance honestly when it surfaces.** Cross-
   validation reports name PASS / PASS-WITH-NOTE / DIVERGENCE
   verdicts per claim:
   - **PASS** = magnitude matches claim exactly OR within ±10pp
   - **PASS-WITH-NOTE** = threshold cleared; magnitude differs >10pp
     from claim but signal direction holds
   - **DIVERGENCE** = magnitude differs substantially (>30pp from
     claim) but threshold may still be cleared OR signal narrowed

3. **DIVERGENCE doesn't trigger doctrinal revision unless threshold
   fails.** If threshold still clears, apparatus suggests arc-driver
   could OPTIONALLY tighten doctrinal text but does NOT unilaterally
   update. The doctrine + measurement instrument + empirical result
   loop stays honest by naming the magnitude-variance, not by forcing
   match.

4. **Future bench-work should expect ±20-30pp variance on N=3.**
   N=5 lifts narrow the variance; cross-judge runs surface judge-
   coupling-bias separately from genuine-substrate-variance. Plan
   bench-shape with this expected variance bound.

5. **Magnitude-variance is NOT a falsification condition.** Per
   `reports/2026-05-09-1330-crown-22-commitment-8-falsification-plan-named.md`,
   falsification conditions are:
   - Pass-rate gap collapse <30pp on E2+E5 DR (gap-collapse, not
     magnitude-shift-while-clearing)
   - Bare unexpectedly passes DR 3/3 broadly
   - Cross-substrate replication collapses signal
   - Real-reader cold-read indistinguishability
   - Rubric tightening collapses gap

   Lower magnitude-while-clearing-threshold is NONE of these.

## Worked example (Sapphire 17 Path B Version C cross-validation 2026-05-10)

| Claim (Version C) | v4 empirical | Verdict | Magnitude delta |
|---|---|---|---|
| E2 Aaron +100pp | +67pp | PASS-WITH-NOTE | -33pp |
| E2 PR +100pp | +100pp | PASS | 0 |
| E5 Aaron +67pp | +33pp | PASS-WITH-NOTE | -34pp |
| E5 PR ceiling-collapse / v3-strict +80pp | +80pp | PASS | 0 |
| E6 Aaron +100pp | +67pp | PASS-WITH-NOTE | -33pp |
| E6 PR +100pp | +100pp | PASS | 0 |
| E7 Aaron +100pp | +100pp | PASS | 0 |
| E7 PR +80-100pp | +40pp | DIVERGENCE | -40 to -60pp |

5 PASS / 3 PASS-WITH-NOTE / 1 DIVERGENCE. ALL 8 cells clear 30pp
threshold. **No falsification condition triggered.** The doctrinal
claim survives; magnitudes-as-stated could be optionally tightened by
arc-driver but apparatus stays at suggestion-only.

The 3 PASS-WITH-NOTE cells share a pattern: Aaron E2/E5/E6 all
read at +33-67pp vs +67-100pp originally. This is Aaron-pipeline-rep3
reading borderline (DR_ext=2 mixed where prior runs read 3 face-value-
holding). N=5 lift on Aaron E2/E5/E6 would narrow the variance.

The 1 DIVERGENCE cell (PR E7 +40pp vs +80-100pp): rep1 + rep4 read
mixed where prior runs read PASS; rep2 still reads endorsed-
allegorization. The cap-scope-guardrail is correctly NOT applying to
pipeline cells per codex 8th note 3; the strictness is canonical-
instrument-correct. The +40pp gap is honest judge-reading.

## Anti-pattern named

**Magnitude-precision-fetish.** Treating magnitude differences between
runs as falsification when threshold-clearing is the load-bearing
verdict gate. This anti-pattern:
- Inflates Sapphire-firing claims beyond what evidence warrants
- Forces spurious re-runs to "match" magnitudes already-stated
- Conflates honest variance with apparatus-discipline failure
- Misuses the cross-validation tool (which exists to surface honest
  variance, not to enforce magnitude-match-precision)
