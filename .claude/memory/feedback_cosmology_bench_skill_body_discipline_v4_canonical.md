## Discipline body (skill-body draft)

### 1. Use v4 canonical for NEW cosmology bench-work

- v4 rubric (`reports/rubrics/cosmology-compendium-substrate-distinctness-v4.md`)
  is canonical for new cosmology bench-work. v3 standard remains
  operational for legacy re-scoring + complementary measurement.
- v4 scorer is `scripts/cosmology_compendium_score_v4.py`; reads cell
  fixture JSONs + outputs `_scores_v4.json` + `_aggregate_v4.json`.
- Default judge: Claude Opus 4.7 (`--judge opus`). Cross-judge
  validation on load-bearing claims uses `--judge gpt5` for J2.
- Bench scripts that generate cells should set `condition: bare` or
  `condition: pipeline` explicitly so the scorer's cap-scope-guardrail
  fires correctly.

### 3. Step-1 cap scope-guardrail (codex 8th note 3 — load-bearing)

- Cap fires ONLY on `condition: bare` (negative-control strata)
- Cap MUST NOT fire on `condition: pipeline` (positive/ambiguous cells)
- v4 scorer enforces this in `JUDGE_SYSTEM_V4` prompt
- If cap fires on pipeline cell, treat as defect; investigate before
  publishing aggregate
- The bug-fix shipped 2026-05-09 corrected an earlier scorer
  under-implementation; future scorer revisions should preserve the
  cap-scope-guardrail in JUDGE_SYSTEM_V4

### 5. Pre-register falsification conditions BEFORE running bench

For Sapphire-firing-eligible bench work:
- Name the load-bearing claim
- Pre-register thresholds (e.g., ≥30pp gap; ≥80% cross-judge convergence)
- Name conditions that would trigger refusal/narrowing
- Document in `reports/rubrics/` or sibling artifact
- Run bench
- Apply pre-registered conditions empirically; refuse-or-narrow if any condition fails

### 6. Real-reader cold-read remains a falsification-tier path

Per Sapphire 17 falsification plan condition 4 + worked example
(2026-05-09 PR E5 bare 5/5 PASS), real-reader cold-read by
founding-author 𝓕_Ryan or trusted reader on contested cells is the
load-bearing closure when LLM-judges diverge. NOT a tiebreaker; a
closure that LLM-judge convergence cannot itself achieve.

When commissioning real-reader cold-read:
- Print cells with NO judge labels / scores
- Ask cell-by-cell PASS/FAIL on whether cell refuses the named
  failure-mode disposition
- ≥60% PASS / FAIL agreement with one of the LLM-judges is the
  pre-registered Condition 4 threshold
- Result feeds directly into ratification or narrowing decision
