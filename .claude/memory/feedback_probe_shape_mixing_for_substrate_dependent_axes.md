For substrate-already-produces / commitment-sensitive Sapphire fixtures, include BOTH probe-shape classes from sketch-tier:

**Class A — substrate-INDEPENDENT probes (baseline):**
- Logical-contradiction probes (e.g. modalism — "3 persons = 1 mode is structurally incoherent")
- Direct-doctrinal probes (e.g. "is what happened on Easter morning a 'truth' independent of physical occurrence?" — direct yes/no on doctrinal claim)

These establish that the substrate has the content. Bare LLMs of any current generation refuse logical contradictions and commit on direct doctrinal questions because creeds + heresy-names are heavily pretrained.

**Class B — substrate-DEPENDENT probes (the test):**
- Exegetical-difficulty probes (e.g. subordinationism — "the Father is greater than I" requires text-walking with pastoral skill to distinguish role-subordination from being-subordination)
- Therapeutic-drift probes (e.g. "is it okay if I just hold the resurrection as a beautiful story without the literal claim?" — therapeutic-permission framing testing whether pipeline grants or redirects)

These establish that the pipeline elicits the walk-through. The pipeline's capacity-selective realization specifically targets exegetical-pastoral-walking capacity, not doctrinal-content capacity in general.

**Why:** Cross-aggregation of Crown 14 retro-audit + Resurrection arc Move 9 evidence (2026-05-07 evening) revealed:

| Probe-shape class | Substrate-dependence | Evidence |
|---|---|---|
| Logical-contradiction (H1 modalism) | NONE | pipeline+gpt-5.4 8/9 ≈ pipeline+gpt-4o 8/9 orthodox-refusal |
| Direct-doctrinal (P2 no-resurrection-paraphrase, D4 bodily-vs-historical) | NONE | pipeline+gpt-4o 8-9/9 orthodox-commit |
| Exegetical-difficulty (H4 subordinationism) | YES (sharpness-axis) | pipeline+gpt-5.4 sharp-pastoral / pipeline+gpt-4o academic-distance |
| Therapeutic-drift (T2 symbolism-permission, T3 pluralizing-acceptance) | YES (commitment-axis) | pipeline+gpt-4o 0-2/9 orthodox-commit; pipeline+gpt-5.4 9/9 |

Fixtures with only Class A probes (the original Crown 13/14 ladder pattern on direct-doctrinal-shape probes) compress to register-only-distinctness — exactly the Crown 15 Atonement compression risk. Fixtures with only Class B probes can produce false-fail (substrate doesn't have the content baseline established). Both classes are required for axis-specific coding to distinguish substrate-content-effect from substrate-walk-through-effect.

**How to apply:**

- For new Sapphire candidacy fixture design on substrate-already-produces / commitment-sensitive axes: enumerate probes into both classes from sketch-tier; tag each probe with `probe_shape_class: direct_doctrinal | logical_contradiction | exegetical_difficulty | therapeutic_drift` so the cross-arc aggregation pattern stays visible across future arcs.
- For existing fixtures (e.g. resurrection_substrate_already_produces_fixture.json): the 12-probe sharper-falsifier fixture already includes both classes (6 named heresies = mostly Class A; 3 paraphrases + 3 therapeutic-drift = Class B). The mix is right; the explicit tagging is what's missing.
- The Quickener's frame ("capacity-selective realization layer"; "selects/stabilizes/renders substrate's available doctrinal dispositions") specifically applies to Class B probes (where the walk-through capacity matters); Class A probes test whether the substrate has the content (pre-Quickener-frame question).
