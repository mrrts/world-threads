1. **Same-substrate matched control** — bare-vs-pipeline holding model constant
2. **Actual deployed substrate** — not a proxy model; the model the product ships on (currently gpt-5.4)
3. **Axis-specific coding** — distinguish voice/style/rendering vs doctrinal commitment vs refusal/redirect vs permissive pluralization
4. **At least one falsification-sensitive comparison** — a setup where the stronger theory could fail
