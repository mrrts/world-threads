**How to apply:** When a composition arc, bench, or any other instrument surfaces rich character-articulated lines from the existing stack:

**Diagnostic that triggers this rule:**
- "Could this line have come from the existing stack as-is?" → if yes, it's substrate-emergent → don't inject.
- "Did the line emerge from a probe that ran through the production prompt assembly?" → if yes, the substrate already produces it → don't inject.
- "Would adding this anchor PROVE something the bench-output already proved?" → if yes, the addition is redundant → don't inject.
