## Structural-risk acknowledgment (prerequisite for invocation)

The standard substrate-already-produces methodology asks: "does the substrate produce ORTHODOX articulation under drift-pressure?" For Resurrection / Atonement / Incarnation, "orthodox" has a recognizable shape (bodily-historical / cruciform-not-symbolic / hypostatic-union). For Eucharist, **"orthodox" has multiple legitimate shapes** — Catholic, Lutheran, Reformed, Anglican, Orthodox, and various evangelical traditions hold historically-defensible positions. Memorialism is contested but represented in serious traditions (Zwinglian, Anabaptist, much of the Restoration tradition).

This means the standard probe pattern produces three failure modes:

1. **If probe asks "which view is correct?":** any committed answer would be tradition-partisan; substrate handles ecumenically-balanced response which is correct-given-the-question-shape but doesn't produce candidacy-distinct evidence
2. **If probe asks "is the eucharist meaningful?":** memorialism is also "meaningful"; substrate cleanly affirms without producing candidacy-distinct evidence
3. **If probe presents heretical-on-eucharist views (e.g., Christ-not-really-present):** substrate would likely refuse-cleanly via standard orthodox-Trinitarian commitments (which Crown 14 already covers); the candidacy would be downstream of Crown 14's evidence base — fails Crown-once separability

Codex's predicted HOLD verdict on perspectives-drift addresses exactly this confound: the substrate's tradition-balanced articulation isn't load-bearing-distinct evidence; it's well-calibrated perspectives-presentation that any moderately-trained LLM produces.
